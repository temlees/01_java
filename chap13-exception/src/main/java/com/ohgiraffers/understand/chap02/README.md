# 영화관 티켓 예매 시스템 구현

Movie  클래스
- 영화 제목, 상영 시간, 총 좌석 수, 예약된 좌석 수 를 속성으로 가짐
- 좌석을 예매하고, 남은 좌석을 확인하는 메소드를 가짐.
- 좌석이 모두 예약된 경우 예매 불가

MovieTheater 클래스
- 여러 개의 Movie 객체를 가짐
- 영화 추가, 영화 목록 조회, 특정 영화 예매 메소드를 가짐

Application 에서 몇 편의 영화를 추가할지 물어보고 영화를 추가한 후
현재 상영 중인 영화 목록을 보여준 다음,

특정 영화 좌석 예매, 상영 중인 영화 목록 조회 중 선택하고 진행
